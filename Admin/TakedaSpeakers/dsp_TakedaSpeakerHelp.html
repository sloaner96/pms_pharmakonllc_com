Overview

Takeda Pharmaceutical has developed a web site portal for its speaker bureau.  All vendors who schedule any of these speakers and provide payment to them as honoraria are required to provide a formatted list of events and speaker data on a regular basis in order to update the Takeda Speaker web portal.  This document describes how to process and deliver these files to Takeda.

 

Frequency

Run every Friday afternoon.

Both the Event Export and the Payment Export procedures should be run every Friday afternoon.

The Takeda Speaker Event export should be backdated as least 2 weeks form the date of program execution and the Takeda Speaker Payment export should be back dated at least 4 weeks from the date of program execution.  This is done to ensure that any changes in event or payment status is captured and overlaid properly on the Takeda Speaker Portal web site.

 

Takeda Speaker Event Processing

  • Log on to PMS
  • Click on the ADMIN menu option
  • Select ‘Export Takeda Speaker Events’ under that Takeda Speaker heading
  • Select the project(s) to export from the Projects drop-down box.  Normally, all projects should be selected.
  • Enter or select the beginning and ending dates of the Takeda events to export.  Notice that in formation about the last export performed is displayed at the top of the page.  Normally, a date at least 2 weeks in the past should be selected for the beginning date and a date at least 2 weeks in the future should be selected for the ending date.
  • Click the Continue button to process the export.
  • After pulling the events based on the criteria entered, a summary of the results will appear.  Any exceptions will be noted on the screen.  All exception must be corrected and the export process re-run before the export file can be produced.
  • If there are no exceptions, click the Export Data button to create the Takeda Events export file. 
  • Click the Finish button to report to the PMS Admin menu.
  • Inform the IT department that the Takeda Speaker Events export file is ready and available for transfer to Takeda.  NOTE: The export file must be encrypted and transferred to Takeda via secure FTP.

 

 

Takeda Speaker Payment Processing

  • Log on to PMS
  • Click on the ADMIN menu option
  • Select ‘Export Takeda Speaker Payments’ under that Takeda Speaker heading
  • Select the project(s) to export from the Projects drop-down box.  Normally, all projects should be selected.
  • Enter or select the beginning and ending dates of the Takeda payments to export.  Notice that in formation about the last export performed is displayed at the top of the page.  Normally, a date at least 4 weeks in the past should be selected for the beginning date and the current date should be selected for the ending date.
  • Click the Continue button to process the export.
  • After pulling the payments based on the criteria entered, a summary of the results will appear.  Any exceptions will be noted on the screen.  All exception must be corrected and the export process re-run before the payment export file can be produced.
  • If there are no exceptions, click the Export Data button to create the Takeda Speaker Payments export file. 
  • Click the Finish button to report to the PMS Admin menu.
  • Inform the IT department that the Takeda Speaker Payments export file is ready and available for transfer to Takeda.  NOTE: The export file must be encrypted and transferred to Takeda via secure FTP.
